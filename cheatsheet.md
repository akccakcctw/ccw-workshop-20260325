## 萬用 Prompt 框架

```
我是 [你的角色]，我需要 [做什麼事]。
我的輸入是 [什麼格式的資料]。
請幫我 [產出什麼結果]，格式要 [怎麼呈現]。
```

### 範例

```
我是行銷主管，我需要分析這份客戶名單。
我的輸入是一份 CSV，包含客戶姓名、國家、消費金額。
請幫我找出消費前 10 名的客戶，格式要用表格呈現。
```

---

## 常用 Prompt 技巧

| 技巧 | 範例 |
|------|------|
| **指定角色** | 「你是一個資深行銷分析師...」 |
| **給範例** | 「格式像這樣：名字 / 國家 / 金額」 |
| **分步驟** | 「第一步...第二步...第三步...」 |
| **指定格式** | 「用表格」「用條列」「用 email 格式」 |
| **限制長度** | 「用 3 句話摘要」「不超過 200 字」 |
| **要求解釋** | 「為什麼這樣建議？」 |

## 追問密技

AI 會記住你之前說的話，所以你可以一直追問：

- 「剛才那份名單，改成只看台灣客戶」
- 「加上上次購買日期的欄位」
- 「幫我翻成英文」
- 「再詳細一點」
- 「太長了，精簡一點」

## 上傳檔案的方法

在 claude.ai 的對話框中，點左下角的 **📎 迴紋針** 圖示，即可上傳：
- CSV / Excel 檔案
- PDF 文件
- 圖片
- 文字檔

上傳後直接說「幫我分析這份資料」就可以了。
