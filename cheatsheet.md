## 萬用 Prompt 框架

```
[情境]：我在準備 [什麼事]，對象是 [誰]。
[輸入]：我有 [什麼格式的資料]。
[目標]：請幫我 [產出什麼結果]，格式要 [怎麼呈現]。
```

### 範例

```
我在準備週會報告，對象是行銷主管。
我有一份客戶名單 CSV，包含姓名、國家、消費金額。
請幫我找出消費前 10 名的客戶，用表格呈現，並附上一段摘要。
```

## 常用 Prompt 技巧

| 技巧 | 範例 |
|------|------|
| **描述情境與目標** | 「我在準備週會報告，需要各管道的 ROAS 比較」 |
| **給範例** | 「格式像這樣：名字 / 國家 / 金額」 |
| **指定格式** | 「用表格」「用條列」「用 email 格式」 |
| **限制長度** | 「用 3 句話摘要」「不超過 200 字」 |
| **追問修改** | 「改成只看台灣客戶」「再精簡一點」 |
| **要求解釋** | 「為什麼這樣建議？」 |

> **小提醒**：你不需要先說「你是一個資深分析師」AI 才會回答得好。
> 現在的 AI 已經很擅長理解意圖——把你的情境和目標講清楚，比指定角色更有效。

## 追問密技

AI 會記住你之前說的話，所以你可以一直追問：

- 「剛才那份名單，改成只看台灣客戶」
- 「加上上次購買日期的欄位」
- 「幫我翻成英文」
- 「再詳細一點」
- 「太長了，精簡一點」

## 你可以請 AI 幫你做的事

- 分析 Excel / CSV 資料
- 寫 Email / 文案（中英文）
- 整理會議記錄
- 比較不同方案的優缺點
- 產生報表摘要
- 檢查文件裡的錯誤
- 翻譯
- 發想行銷活動點子

## 上傳檔案的方法

在 Claude Desktop 的對話框中，點 **📎 迴紋針** 圖示（或直接拖曳檔案到對話框），即可上傳：
- CSV / Excel 檔案
- PDF 文件
- 圖片
- 文字檔

上傳後直接說「幫我分析這份資料」就可以了。
